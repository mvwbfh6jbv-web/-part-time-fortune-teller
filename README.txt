FOLDED ORACLE — OPTION 1 DEPLOYMENT

Upload all three files in this folder to the ROOT of your GitHub Pages repository:

- index.html    = Version 3.0 (this becomes the live homepage)
- version2.html = Original Version 2.0 backup
- README.txt    = these instructions

Important:
1. Make sure the filenames stay exactly lowercase as shown.
2. Replace the old root index.html with this new index.html.
3. Do not upload the enclosing folder itself if GitHub puts it inside another folder.
4. After GitHub Pages finishes publishing, open your normal site address.
5. To view the original backup, add /version2.html to the end of your site address.

Example:
https://yourname.github.io/version2.html

GitHub Pages may take a minute or two to update after the upload.
